# 1.　はじめに
利用するためには、BPMクレジット決済サービスに申し込みが必要となります。

**申込みはこちらから(https://bpmc.co.jp/)**

# 2.　プラグインの概要
WoocommerceにてBPMクレジット決済をするためのプラグインになります。

# 3.　サポート環境
- WordPress v6.4.1
- Woocommerce v8.3.0
- PHP v8.2.9
